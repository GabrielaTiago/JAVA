# Cars API

## Pemissa

API Spring para carros. Rota POST **"/api/cars"** para realizar a inserção de um novo carro.

Utlizando conceitos de controladores e DTO

```json

    body:

    {
        "modelo": "Fiat UNO 1.3",
        "fabricante": "Fiat",
        "dataFabricacao": "10/02/2006",
        "valor": 13.000,
        "anoModelo": 2006
    }
```
