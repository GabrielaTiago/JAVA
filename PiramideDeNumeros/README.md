# Pirâmide de números

## Premissa

Código recebe dois valores numéricos, e imprime uma prirâmide de números. Seprando em cada linha, a quantidade de números que estão sendo iterados no momento.

```java
    Primeiro número: 1
    Segundo número: 5

    Resultado:

    1
    22
    333
    4444
    55555
```

```java
    Primeiro número: 5
    Segundo número: 9

    Resultado:

    55555
    666666
    7777777
    88888888
    999999999
```
